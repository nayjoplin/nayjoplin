# AI Beauty Product Recommendation

Projeto de recomendação de produtos de beleza com base em perfil, necessidades, objetivos e comportamento do usuário, conectando dados, produto digital, experiência do usuário e inteligência artificial.

## Objetivo

Criar uma solução capaz de sugerir produtos de beleza a partir de informações do usuário, como tipo de cabelo, objetivo, histórico de compra, preferências e necessidade principal.

## Problema de negócio

Consumidores de beleza frequentemente têm dificuldade para escolher produtos adequados. Marcas e e-commerces podem melhorar conversão, experiência e retenção ao oferecer recomendações mais inteligentes e personalizadas.

## Funcionalidades previstas

- Cadastro de produtos.
- Cadastro de atributos dos produtos.
- Questionário de perfil do usuário.
- Lógica de recomendação baseada em regras.
- API para consultar recomendações.
- Interface simples para simular experiência do usuário.
- Documentação técnica da solução.

## Variáveis sugeridas

- Tipo de cabelo.
- Objetivo principal.
- Necessidade específica.
- Histórico de compra.
- Preferência por categoria.
- Produto já utilizado.
- Ativos do produto.
- Benefícios prometidos.
- Frequência de uso.

# Beauty Sales Intelligence Dashboard

Projeto de analytics aplicado ao setor de beleza e e-commerce, com foco em vendas, produtos, canais, comportamento de compra e oportunidades de crescimento.

## Objetivo

Criar um dashboard de inteligência comercial para marcas de beleza, conectando dados de vendas, categorias, canais e performance para apoiar decisões de marketing, produto e growth.

## Problema de negócio

Marcas de beleza precisam entender quais produtos vendem melhor, quais canais performam mais, como o ticket médio evolui e quais oportunidades existem para aumentar conversão, recompra e receita.

## Funcionalidades previstas

- Organização de base de vendas simulada.
- Análise de receita por produto.
- Análise de ticket médio.
- Análise de produtos mais vendidos.
- Comparação entre canais de venda.
- Visualização de evolução mensal.
- Identificação de oportunidades comerciais.
- Dashboard executivo em Power BI.

## Indicadores sugeridos

- Receita total.
- Receita por categoria.
- Ticket médio.
- Quantidade de pedidos.
- Produtos mais vendidos.
- Conversão por canal.
- Evolução mensal de vendas.
- Participação por linha de produto.
- Clientes recorrentes.

# Financial Data Pipeline

Projeto de engenharia de dados aplicado ao mercado financeiro, com foco em coleta, tratamento, transformação, armazenamento e visualização de dados.

## Objetivo

Construir um pipeline de dados financeiros para transformar dados brutos em informações estruturadas, prontas para análise, dashboard e tomada de decisão.

## Problema de negócio

Empresas financeiras lidam com grande volume de dados vindos de diferentes fontes. Um pipeline bem estruturado permite organizar dados, reduzir inconsistências, automatizar análises e gerar indicadores confiáveis.

## Funcionalidades previstas

- Coleta de dados financeiros simulados.
- Tratamento de dados brutos.
- Padronização de campos.
- Criação de processo ETL.
- Armazenamento em banco relacional.
- Geração de tabelas analíticas.
- Dashboard com indicadores.
- Documentação de arquitetura.

## Fontes de dados sugeridas

- Transações financeiras.
- Clientes.
- Produtos financeiros.
- Carteiras de investimento.
- Canais de atendimento.
- Métricas de receita e performance.

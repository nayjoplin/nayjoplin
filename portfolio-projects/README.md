# Portfólio de Projetos | Tecnologia, Dados, Finanças e Beauty Tech

Este diretório reúne projetos estratégicos criados para demonstrar competências em engenharia de software, dados, analytics, produto digital, growth e negócios.

Os projetos foram desenhados para conectar minha experiência como Coordenadora de Marketing e Growth com minha formação em Engenharia da Computação, Bacharelado em Tecnologia da Informação e Análise e Desenvolvimento de Sistemas.

## Objetivo do portfólio

- Demonstrar domínio de lógica, dados, APIs, dashboards e produto digital.
- Criar projetos alinhados a bancos, fintechs, consultorias estratégicas, investimentos e empresas de tecnologia.
- Unir visão técnica com experiência real em growth, marketing analytics, gestão de projetos e coordenação de equipes multidisciplinares.
- Construir evidências práticas para processos seletivos em empresas como Itaú, BTG Pactual, XP Inc., J.P. Morgan, Goldman Sachs, McKinsey, BCG, Bain e empresas de tecnologia.

## Organização sugerida de cada projeto

Cada projeto deve conter:

- README com visão de negócio, escopo técnico e diferenciais.
- Base de dados ou dados simulados.
- Código-fonte organizado.
- Documentação de arquitetura.
- Prints ou links de dashboards/interfaces.
- Roadmap de evolução.
- Próximos passos técnicos.

# Personal Investment Portfolio Analyzer

Projeto para análise de carteira de investimentos, com foco em dados financeiros, visualização de performance, risco, diversificação e tomada de decisão.

## Objetivo

Criar uma ferramenta analítica para organizar ativos, aportes, rentabilidade, composição de carteira e indicadores de performance de investimentos.

## Problema de negócio

Investidores precisam acompanhar evolução patrimonial, diversificação, exposição ao risco e desempenho dos ativos. Este projeto simula uma solução de análise para apoiar decisões financeiras.

## Funcionalidades previstas

- Cadastro de ativos.
- Registro de aportes.
- Cálculo de rentabilidade.
- Análise de composição da carteira.
- Comparação entre classes de ativos.
- Visualização de evolução patrimonial.
- Simulação de cenários.
- Dashboard executivo.

## Indicadores sugeridos

- Patrimônio total.
- Rentabilidade acumulada.
- Rentabilidade mensal.
- Distribuição por classe de ativo.
- Concentração por ativo.
- Aportes por período.
- Evolução da carteira.
- Comparação entre ativos.
